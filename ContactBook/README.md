# ContactBook 聯絡人管理系統

一個基於 Qt C++ 開發的聯絡人管理應用程式，提供完整的聯絡人增刪查改功能。

## 功能特色

### 核心功能
- ✅ **新增聯絡人**：輸入學號、班級、姓名、電話資訊
- ✅ **編輯聯絡人**：選擇現有聯絡人進行編輯
- ✅ **刪除聯絡人**：刪除不需要的聯絡人資料
- ✅ **搜尋功能**：快速搜尋符合關鍵字的聯絡人
- ✅ **匯入檔案**：從 TXT/CSV 檔案匯入聯絡人
- ✅ **匯出檔案**：將聯絡人資料匯出至檔案

### 優化項目
- 🎨 **美化介面**：使用 Fusion 樣式，添加群組框和提示文字
- 🔒 **輸入驗證**：確保所有欄位不為空才能新增/編輯
- 💬 **使用者反饋**：狀態列訊息和確認對話框
- 📊 **表格功能**：支援排序、交替行顏色、自動調整欄寬
- 🗂️ **智慧路徑**：使用系統文件夾作為預設儲存位置
- ✨ **清除功能**：一鍵清除輸入表單
- 🔍 **即時搜尋**：搜尋時隱藏不符合的項目

### 使用者體驗提升
- 所有按鈕都有工具提示
- 確認對話框防止誤操作
- 狀態列顯示操作結果
- 選擇表格項目自動填入表單
- 輸入欄位有佔位符提示
- 錯誤訊息友善易懂

## 系統需求

- Qt 6.x 或更高版本
- C++17 或更高版本
- 支援 Windows、macOS、Linux

## 建置方式

### 使用 Qt Creator
1. 開啟 `ContactBook.pro` 專案檔
2. 選擇適合的 Kit
3. 點擊建置並執行

### 使用命令列
```bash
qmake ContactBook.pro
make
./ContactBook
```

## 檔案格式

匯入/匯出的檔案格式為 CSV（逗號分隔）：
```
學號,班級,姓名,電話
12345,資工一甲,王小明,0912345678
12346,資工一乙,李小華,0923456789
```
