# ContactBook - 通訊錄管理系統

一個功能完整的通訊錄管理應用程式，使用 Qt C++ 開發。

## ✨ 主要功能

### 基本功能
- ✅ **新增聯絡人** - 快速新增學生聯絡資訊
- ✅ **編輯聯絡人** - 修改現有聯絡人資料
- ✅ **刪除聯絡人** - 移除不需要的聯絡人
- ✅ **清除表單** - 快速清空輸入欄位

### 資料管理
- 📥 **匯入資料** - 從 TXT/CSV 檔案匯入聯絡人
- 📤 **匯出資料** - 將聯絡人匯出為 TXT/CSV 檔案
- 🔍 **即時搜尋** - 快速搜尋學號、班級、姓名或電話
- 🔄 **欄位排序** - 點擊欄位標題進行排序

### 使用者體驗
- ✅ **輸入驗證** - 自動檢查必填欄位和格式
- 💬 **確認對話框** - 重要操作前的確認提示
- 📊 **即時計數** - 狀態列顯示聯絡人總數
- ⌨️ **鍵盤快捷鍵** - 提升操作效率
- 🎨 **美化介面** - 清晰的視覺設計和交替行色

## 🖥️ 系統需求

- Qt 6.x 或更高版本
- C++17 支援的編譯器
- Windows / Linux / macOS

## 🚀 安裝與編譯

### 使用 Qt Creator
1. 開啟 Qt Creator
2. 載入 `ContactBook.pro` 專案檔
3. 選擇適當的 Kit
4. 點擊「建置」按鈕
5. 執行程式

### 使用命令列
```bash
qmake ContactBook.pro
make
./ContactBook  # Linux/macOS
# 或
ContactBook.exe  # Windows
```

## 📋 使用說明

### 新增聯絡人
1. 在輸入欄位填寫聯絡人資訊
   - **學號**（必填）
   - **班級**（選填）
   - **姓名**（必填）
   - **電話**（選填）
2. 點擊「新增」按鈕或按 Enter 鍵

### 編輯聯絡人
1. 在表格中選擇要編輯的聯絡人
2. 點擊「編輯」按鈕
3. 修改資料後點擊「新增」完成更新

### 刪除聯絡人
1. 在表格中選擇要刪除的聯絡人
2. 點擊「刪除」按鈕或按 Del 鍵
3. 確認刪除操作

### 搜尋聯絡人
- 在搜尋欄位輸入關鍵字
- 系統會自動篩選符合的聯絡人
- 支援搜尋所有欄位（學號、班級、姓名、電話）

### 匯入/匯出資料
- **匯入**: 點擊「匯入」按鈕選擇檔案 (Ctrl+O)
- **匯出**: 點擊「匯出」按鈕選擇儲存位置 (Ctrl+S)
- 支援格式: TXT, CSV

## ⌨️ 鍵盤快捷鍵

| 功能 | 快捷鍵 |
|------|--------|
| 新增聯絡人 | Enter |
| 刪除聯絡人 | Del |
| 清除表單 | Esc |
| 匯入檔案 | Ctrl+O |
| 匯出檔案 | Ctrl+S |
| 結束程式 | Ctrl+Q |

## 📁 檔案格式

### CSV/TXT 格式
```
學號,班級,姓名,電話
S001,1A,張三,0912-345-678
S002,1B,李四,0923-456-789
```

每行一筆資料，欄位間以逗號分隔。

## 🎨 特色功能

- **交替行色**: 提高表格可讀性
- **欄位排序**: 點擊欄位標題即可排序
- **輸入驗證**: 防止無效資料輸入
- **狀態提示**: 即時顯示操作結果
- **工具提示**: 滑鼠懸停顯示功能說明
- **確認對話框**: 防止誤操作

## 🔧 技術細節

- **框架**: Qt 6.9.1
- **語言**: C++17
- **UI 設計**: Qt Designer (.ui 檔案)
- **架構**: Model-View 模式
- **資料格式**: CSV (逗號分隔)
